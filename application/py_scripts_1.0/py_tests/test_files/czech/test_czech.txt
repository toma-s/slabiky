Česko je země s tržním hospodářstvím, která podle ekonomických, sociálních a politických indikátorů, jako je index lidského rozvoje, index svobody tisku, index svobody internetu od cenzury, patří k vysoce hodnoceným státům světa. Míra nezaměstnanosti je dlouhodobě nízká a pod průměrem vyspělých zemí.
